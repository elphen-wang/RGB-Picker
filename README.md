# PDF/Image Heatmap Picker

## Abstract
PDF/Image Heatmap Picker is a Tkinter GUI app to load PDFs or images, display pages, and pick points to extract RGB colors. It maps colors to heatmap values using customizable colormaps and ranges. Users can save collected data points to TXT. Ideal for analyzing color-coded heatmaps visually.

## Introduction 

This is a GUI application built with Tkinter and Matplotlib that allows users to load and view PDF pages or images. Users can interactively pick points on the displayed content to extract RGB color information. The app then maps these RGB colors to numerical heatmap values based on a chosen colormap and value range. Key features include:

* Load PDF files and navigate pages.
* Load common image formats (PNG, JPG, JPEG, BMP, GIF).
* Display the selected PDF page or image with adjustable DPI.
* Interactive picking mode to select points and record their color data.
* Convert picked RGB colors to heatmap values using customizable colormaps and value ranges.
* Save collected data points (coordinates, RGB, heatmap value) to a TXT file.
* Responsive UI with dynamic control layout and real-time info display.
This tool is useful for analyzing color-coded heatmaps embedded in PDFs or images and extracting quantitative data from visual representations.

## 中文简介

这是一个基于Tkinter和Matplotlib的图形界面程序，支持加载PDF文档或常见图片格式，显示指定页面或图片。用户可以在显示内容上点击采集像素点的颜色信息，程序根据所选的颜色映射（colormap）和数值范围，计算对应的热力值。主要功能包括：

* 加载PDF文件并支持翻页查看。
* 加载PNG、JPG、JPEG、BMP、GIF等图片格式。
* 支持调整显示DPI，清晰展示页面或图片。
* 交互式“拾取模式”，点击图片采集RGB颜色点。
* 根据自定义颜色映射和数值区间，将RGB颜色转换为热力图对应数值。
* 支持将采集的点数据（坐标、RGB值、热力值）保存为TXT文本文件。
* 界面布局自适应，实时显示采集信息。
该工具适合从PDF或图片中的热力图视觉效果提取定量数据，方便科研、工程等领域的颜色数据分析。
